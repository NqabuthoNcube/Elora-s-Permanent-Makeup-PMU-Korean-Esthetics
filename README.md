# Elora-s-Permanent-Makeup-PMU-Korean-Esthetics
A basic HTML, CSS and Javascript webpage assignment.
I built a mobile-first single-page landing page in vanilla HTML/CSS/JS (Plain Core web development standards) to keep it fast and dependency-free. The design takes insight from Elora's Korean aesthetic warm rose tones, Cormorant Garamond typography while keeping conversion front and centre through a sticky WhatsApp CTA (Call to Action buttons). SEO is handled via semantic H1/H2 structure, optimised meta tags, and a Local Business schema markup for the Delhi location.

I managed to Link WhatsApp CTA and Calling CTA with the actual Elora's Korean's contact number and kept dummy images in the assets file.
